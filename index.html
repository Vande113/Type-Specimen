Merriweather
Merriweather is a font created by Eben Sorkin with the goal of being a typeface for screens and the web that is as legible and pleasant to read as possible regardless of the design of the screen or web browser it’s on. Merriweather was inspired by many 15th and 16th-century fonts such as Garamonds, Jannons, and Italian type. Notable features of the typeface include its large X height, horizontal compression, vertical stems with no entasis, and diagonal stress instead of vertical stress like most fonts have. While all of these can make Merriweather look a bit odd for print, that’s because it was designed for screens first, and print second.
